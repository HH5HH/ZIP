ZIP

ZIP = Zeek Info Peek — ZIP into INFO from the Zendesk API.

Legacy standalone SPA flow has been retired. ZIP is now packaged as a pure Chrome extension with no third-party dependencies.

One-time setup uses Chrome extension loading. After that, daily usage is under 3 clicks: open Zendesk, click ZIP, login if prompted.
