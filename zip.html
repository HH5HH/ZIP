ZIP = Zeek Info Peek — ZIP into INFO from the Zendesk API.

ZIP is packaged as a pure Chrome extension with no third-party dependencies. One-time setup is handled in chrome://extensions, then daily use is: open Zendesk, open ZIP, login if prompted.
