Defect Severity Guidance

  • P1: Cannot open ZIP or cannot load tickets.
  • P2: Wrong dataset, wrong navigation target, broken CSV fidelity.
  • P3: Visual/layout issue with workaround.
  • P4: Cosmetic text/spacing issue.
